Definition. A functor F:J→D is said to be cofinal if

•

For every object A∈D there is an object B∈J with a morphism f:A→F(B).

•

For any two such morphisms f,g, there are two more morphisms in J such that the following commutes:AF(B)F(C)F(D)gfF(u)F(v)
